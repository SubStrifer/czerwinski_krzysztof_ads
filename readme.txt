### Rebuilding game on Windows using nmake
1. Open "Developer Command Prompt for VS".
2. Move to the repo folder using "cd".

#### Compiling
3. Type "nmake" and confirm with Enter.
4. Makefile will clear and recompile all files, .exe can be found in the executable/ folder.

#### Compiling and running
3. Type "nmake run" and confirm with Enter.
4. Makefile will run the game in the terminal window immediately after compiling.
